# Buking Backend

A Spring Boot backend for property booking management.

## Project Structure
- `src/main/java` - Source code
- `src/main/resources` - Configuration and templates
- `src/test/java` - Unit tests

## Getting Started
1. Clone the repository
2. Build with Maven:
   ```bash
   mvn clean install
   ```
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```
